# mnoclient

## env config
```
export MINIO_SERVER=xxxxx
export AWS_ACCESS_KEY_ID=xxxxx
export AWS_SECRET_ACCESS_KEY=xxxxx
```

## help
```
$ mnoclient -h
Usage of mnoclient: 
  -bucket string
        bucket name
  -download string
        download file path
  -file string
        file name
```

## execute
```
mnoclient -bucket xxx -file xxx -download xxx/xxxx
```
